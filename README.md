# snake-reinforcement-learning

Snake game AI using deep Q-networks

## Training

Example:
```
python train.py --level_filepath levels/9x9_empty.yml
```

## Playing

Example:
```
python play.py --level_filepath levels/9x9_empty.yml --checkpoint best
```
